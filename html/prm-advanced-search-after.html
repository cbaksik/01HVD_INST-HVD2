The HOLLIS User Guide has numerous tips & tricks on using HOLLIS. The page on Advanced Search explains the precision search options and provides numerous examples. Some quick tips:

  • Language: Search for all titles in a language by entering the 3-character language code into the field named 'Code: MARC langauge'
  • Place of publication: Search for all titles published in a particular place by entering the 3-character code into the field named 'Code: MARC place of pub'
  • Starts with: The "starts with" option can only be used with Title or Holding call number searches
  • Audience: Intended audience -- example: “People with visual disabilities”
  • Call number: Use 'Holding call number' -- this will search all call number schemas (Library of Congress, Old Widener, ...); include spaces where appropriate (and if you don't find what you're looking for, try removing spaces)
  • Binding, ownership: search for copy-specific information such as "marbled boards" by using the 'Holding notes on binding, ownership' field
