The HOLLIS User Guide has numerous tips & tricks on using HOLLIS. The page on Advanced Search explains the precision search options and provides numerous examples. Some quick tips:

  • You can use "starts with" for Title or Holding call number searches only.
  • Audience -- example: “People with visual disabilities”
  • Holding call number -- this will search all call number schemas (Library of Congress, Old Widener, ...); include spaces where appropriate (and if you don't find what you're looking for, try removing spaces)
  • Holding notes on binding, ownership -- search for copy-specific information such as "marbled boards"
